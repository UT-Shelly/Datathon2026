# Datathon2026

## Overview
This project analyzes the key factors for financial resilience, and what factors are signs that an individual or family is thriving, coping, or at risk of financial stress. 

## Installation
Clone this repository:
   ```bash
   git clone https://github.com/JYW5114/TTC_delay.git

## Usage
Run the Jupyter Notebook to analyze subway delay data:
```bash
jupyter notebook personal_finance_analysis.ipynb
```

### Data
- **personal_finance_dataset.xslx**: Contains subway delay records with columns like Date, Time, Station, Delay Duration, Line, and Vehicle.

### Key Features
- Setup and Load Data
- Cleaning the Dataset
- Statistical Analysis & Modeling to Model Financial Stress and Stability
